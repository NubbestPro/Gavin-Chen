Project Information
ISS and Tiangong


This project tracks the positions of the International Space Station (ISS), as well as the Tiangong Space Station. Listed below each space station are the astronauts currently on each station.
The dates and local times of cities are also shown. Yellow represents Houston, green represents Toronto, and red represents Beijing.


Code Craft


Welcome to Code Craft! In this sandbox game, you can create whatever you want, however you want. Use the arrow keys or WASD to move, craft new items based on the rules found on the
right side of the map and use the space bar to collect the resource that you are standing on. Have fun!


Snowflakes


This is a Python code that randomly draws six- or eight-branched snowflakes. The size, colour, and position of each snowflake are also randomized.
